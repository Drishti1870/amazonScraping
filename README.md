# Amazon Scraping Project

This project scrapes Amazon search results using Selenium and extracts
product details using BeautifulSoup.

## Features
- Pagination support
- Extracts product title, price, and product link
- Stores raw HTML locally
- Converts data into CSV using Pandas

## Tech Stack
- Python
- Selenium
- BeautifulSoup
- Pandas
